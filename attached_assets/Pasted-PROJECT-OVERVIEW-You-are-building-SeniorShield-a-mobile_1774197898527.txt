PROJECT OVERVIEW

You are building SeniorShield, a mobile app (iOS/Android) that helps seniors use their phones with confidence while protecting them from scams. The app uses a Guide-Not-Controller model — it provides patient, step-by-step voice guidance without automating or controlling the phone.

Product Name: SeniorShield
Tagline: "Your parent's voice assistant for tech help and scam protection"
Core Philosophy: SeniorShield guides seniors through tasks. It does NOT automate or control the phone. Seniors remain in control at all times.

Target Users:

•
Primary: Seniors 65+ who need help with their phones

•
Secondary: Adult children who want to protect their parents

•
Tertiary: Senior center staff who support residents

Revenue Model:

•
B2C: Free tier (voice assistance) + Premium ($12.99/mo or $129.99/yr)

•
B2B: Senior centers ($29.99-50/user/mo)

•
Partnerships: Verizon, AT&T, Medicare Advantage (included in MVP)

Timeline: 12-14 weeks to MVP launch (iOS/Android app stores)
Includes: All 7 critical gaps (onboarding, support, error handling, voice quality, scam accuracy, security, analytics)

DATABASE SCHEMA

Create the following PostgreSQL tables:

1. Users Table

SQL


CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  phone_number VARCHAR UNIQUE,
  email VARCHAR UNIQUE NOT NULL,
  password_hash VARCHAR NOT NULL,
  user_type VARCHAR NOT NULL DEFAULT 'senior', -- 'senior', 'adult_child', 'staff'
  first_name VARCHAR,
  last_name VARCHAR,
  date_of_birth DATE,
  preferred_voice VARCHAR DEFAULT 'female', -- 'female', 'male'
  voice_speed DECIMAL DEFAULT 1.0, -- 0.5 to 1.5
  voice_volume DECIMAL DEFAULT 0.8, -- 0.0 to 1.0
  color_scheme VARCHAR DEFAULT 'light', -- 'light', 'dark', 'warm', 'cool'
  high_contrast_enabled BOOLEAN DEFAULT FALSE,
  font_size VARCHAR DEFAULT 'normal', -- 'small', 'normal', 'large', 'extra_large'
  font_family VARCHAR DEFAULT 'sans-serif', -- 'sans-serif', 'serif'
  line_height VARCHAR DEFAULT 'normal', -- 'compact', 'normal', 'spacious'
  letter_spacing VARCHAR DEFAULT 'normal', -- 'normal', 'wide'
  haptic_feedback BOOLEAN DEFAULT TRUE,
  screen_reader BOOLEAN DEFAULT FALSE,
  hearing_aid_connected BOOLEAN DEFAULT FALSE,
  hearing_aid_model VARCHAR,
  captions_enabled BOOLEAN DEFAULT TRUE,
  data_collection_enabled BOOLEAN DEFAULT TRUE,
  location_access BOOLEAN DEFAULT FALSE,
  microphone_access BOOLEAN DEFAULT TRUE,
  onboarding_completed BOOLEAN DEFAULT FALSE,
  onboarding_step INT DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);



2. User Tiers Table

SQL


CREATE TABLE user_tiers (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id) ON DELETE CASCADE,
  tier VARCHAR NOT NULL DEFAULT 'free', -- 'free', 'premium_trial', 'premium_paid'
  trial_start_date DATE,
  trial_end_date DATE,
  premium_start_date DATE,
  premium_end_date DATE,
  billing_cycle VARCHAR, -- 'monthly', 'annual'
  stripe_subscription_id VARCHAR,
  status VARCHAR DEFAULT 'active', -- 'active', 'expired', 'canceled'
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(user_id)
);

BACKEND API ENDPOINTS

Build the following API endpoints in Express:

Authentication

Plain Text


POST /api/auth/signup
- Input: email, password, user_type (senior, adult_child, staff)
- Output: user_id, token, user_type
- Logic: Create user, hash password, return JWT token

POST /api/auth/login
- Input: email, password
- Output: user_id, token, user_type
- Logic: Verify credentials, return JWT token

POST /api/auth/logout
- Input: token
- Output: success
- Logic: Invalidate token

GET /api/auth/verify
- Input: token
- Output: user_id, email, user_type
- Logic: Verify token validity



User Profile

Plain Text


GET /api/user/profile
- Input: user_id (from token)
- Output: user object (name, email, preferences)
- Logic: Return user data

PUT /api/user/profile
- Input: user_id, first_name, last_name, phone_number
- Output: updated user object
- Logic: Update user profile

GET /api/user/preferences

- Logic: Return all user preferences

PUT /api/user/preferences

- Output: updated preferences
- Logic: Update user preferences



Voice Assistance

Plain Text


POST /api/voice/process-request
- Input: user_id, audio_file (speech-to-text)
- Output: request_text, response_text, task_category, success, error_message
- Logic:
  1. Convert speech to text using OpenAI Realtime API
  2. Classify request (safe vs. escalate)
  3. Generate response using OpenAI GPT
  4. Store in voice_assistance_history
  5. Log any errors to error_logs
  6. Return response with error handling

POST /api/voice/generate-speech
- Input: user_id, text
- Output: audio_file (text-to-speech)
- Logic:
  1. Get user's voice preferences (gender, speed, volume)
  2. Generate speech using OpenAI TTS
  3. Apply user's speed and volume preferences
  4. Return audio file
  5. Log errors if generation fails

GET /api/voice/history
- Input: user_id
- Output: array of voice assistance history
- Logic: Return last 50 voice requests

POST /api/voice/test
- Input: user_id
- Output: audio_file (test speech)
- Logic: Generate test speech for voice quality testing

POST /api/voice/feedback
- Input: user_id, request_id, feedback (quality, clarity, speed)
- Output: success
- Logic: Store voice feedback for improvement



Scam Analyzer

Plain Text


POST /api/scam/analyze
- Input: user_id, screenshot_file (or extracted_text)
- Output: risk_score, risk_level, analysis_details, explanation
- Logic:
  1. Extract text from screenshot using Tesseract/AWS Textract
  2. Check against scam library
  3. Generate risk score (0-100)
  4. Classify risk level (safe, suspicious, high_risk)
  5. Store in scam_analysis table
  6. If high_risk, notify family and staff
  7. Return analysis with explanation

GET /api/scam/history
- Input: user_id
- Output: array of scam analyses
- Logic: Return last 20 scam analyses

GET /api/scam/library
- Input: none
- Output: array of scam library entries
- Logic: Return all scams in library

POST /api/scam/feedback
- Input: user_id, scam_analysis_id, feedback_type (correct, false_positive, false_negative), explanation
- Output: success
- Logic: Store feedback for model improvement and accuracy tracking

POST /api/scam/appeal
- Input: user_id, scam_analysis_id, reason
- Output: success, appeal_status
- Logic:
  1. Create appeal record
  2. Notify admin
  3. Return appeal status

// Scam analysis with accuracy tracking
async function analyzeScam(extractedText) {
  try {
    // Get scam library
    const scamLibrary = await getScamLibrary();
    
    let riskScore = 0;
    const detectedPatterns = [];
    
    // Check for common scam keywords
    const phishingKeywords = ['verify', 'confirm', 'urgent', 'click here', 'update', 'account'];
    const urgencyKeywords = ['immediately', 'urgent', 'act now', 'limited time', 'expires'];
    const moneyKeywords = ['payment', 'wire', 'gift card', 'money', 'cash'];
    const personalInfoKeywords = ['ssn', 'social security', 'password', 'pin', 'credit card'];
    
    if (phishingKeywords.some(keyword => extractedText.toLowerCase().includes(keyword))) {
      riskScore += 30;
      detectedPatterns.push('phishing_keywords');
    }
    
    if (urgencyKeywords.some(keyword => extractedText.toLowerCase().includes(keyword))) {
      riskScore += 20;
      detectedPatterns.push('urgency');
    }
    
    if (moneyKeywords.some(keyword => extractedText.toLowerCase().includes(keyword))) {
      riskScore += 25;
      detectedPatterns.push('money_request');
    }
    
    if (personalInfoKeywords.some(keyword => extractedText.toLowerCase().includes(keyword))) {
      riskScore += 25;
      detectedPatterns.push('personal_info_request');
    }
    
    // Check for suspicious patterns
    if (hasSuspiciousLinks(extractedText)) {
      riskScore += 15;
      detectedPatterns.push('suspicious_links');
    }
    
    if (hasExcessiveMisspellings(extractedText)) {
      riskScore += 10;
      detectedPatterns.push('misspellings');
    }
    
    if (impersonatesBank(extractedText)) {
      riskScore += 20;
      detectedPatterns.push('bank_impersonation');
    }
    
    // Determine risk level
    let riskLevel = 'safe';
    if (riskScore >= 70) {
      riskLevel = 'high_risk';
    } else if (riskScore >= 40) {
      riskLevel = 'suspicious';
    }
    
    return {
      success: true,
      risk_score: Math.min(riskScore, 100),
      risk_level: riskLevel,
      detected_patterns: detectedPatterns,
      analysis_details: {
        phishing_detected: riskScore >= 30,
        urgency_detected: riskScore >= 20,
        money_request_detected: riskScore >= 25,
        personal_info_request_detected: riskScore >= 25,
      }
    };
  } catch (error) {
    console.error('Scam analysis error:', error);
    return { success: false, error: 'Could not analyze screenshot. Please try again.' };
  }
}

// Helper functions
function hasSuspiciousLinks(text) {
  const urlPattern = /https?:\/\/[^\s]+/g;
  const urls = text.match(urlPattern ) || [];
  return urls.some(url => url.includes('bit.ly') || url.includes('tinyurl') || url.includes('short.link'));
}

function hasExcessiveMisspellings(text) {
  // Simple check: count words that don't match dictionary
  // In production, use a proper spell checker
  const words = text.split(/\s+/);
  const misspelledCount = words.filter(word => word.length > 3 && !isValidWord(word)).length;
  return misspelledCount > words.length * 0.1; // More than 10% misspelled
}

function impersonatesBank(text) {
  const bankNames = ['bank', 'paypal', 'amazon', 'apple', 'microsoft', 'google'];
  return bankNames.some(bank => text.toLowerCase().includes(bank));
}

function isValidWord(word) {
  // In production, use a proper dictionary API
  return word.length > 0;
}

// Data encryption
const crypto = require('crypto');

function encryptData(data, encryptionKey) {
  const iv = crypto.randomBytes(16);
  const cipher = crypto.createCipheriv('aes-256-cbc', Buffer.from(encryptionKey), iv);
  let encrypted = cipher.update(data);
  encrypted = Buffer.concat([encrypted, cipher.final()]);
  return iv.toString('hex') + ':' + encrypted.toString('hex');
}

function decryptData(data, encryptionKey) {
  const parts = data.split(':');
  const iv = Buffer.from(parts.shift(), 'hex');
  const encryptedText = Buffer.from(parts.join(':'), 'hex');
  const decipher = crypto.createDecipheriv('aes-256-cbc', Buffer.from(encryptionKey), iv);
  let decrypted = decipher.update(encryptedText);
  decrypted = Buffer.concat([decrypted, decipher.final()]);
  return decrypted.toString();
}
