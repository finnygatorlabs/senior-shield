Step 2: Database Schema

Create these tables in PostgreSQL:

SQL


-- User profiles table
CREATE TABLE user_profiles (
  user_id VARCHAR(255) PRIMARY KEY,
  name VARCHAR(255) NOT NULL,
  interests JSONB DEFAULT '[]',
  family_members JSONB DEFAULT '[]',
  onboarding_complete BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Progressive data collection table
CREATE TABLE conversation_data (
  id SERIAL PRIMARY KEY,
  user_id VARCHAR(255) NOT NULL REFERENCES user_profiles(user_id),
  data JSONB NOT NULL,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Create indexes for performance
CREATE INDEX idx_user_profiles_user_id ON user_profiles(user_id);
CREATE INDEX idx_conversation_data_user_id ON conversation_data(user_id);
CREATE INDEX idx_conversation_data_created_at ON conversation_data(created_at);



Step 3: Integration with App

In your App.tsx:

TSX


import { FastTrackOnboarding } from './screens/SeniorShield-FastTrack-Onboarding';

export default function App() {
  const [user, setUser] = useState(null);
  const [onboardingComplete, setOnboardingComplete] = useState(false);

  const handleOnboardingComplete = async (data) => {
    // Save to backend
    const response = await fetch('/api/onboarding/complete', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        userId: user.id,
        ...data
      })
    });

    const result = await response.json();
    setOnboardingComplete(true);
    // Navigate to chat screen
  };

  const handleOnboardingSkip = () => {
    // Use defaults and go to chat
    setOnboardingComplete(true);
  };

  if (!onboardingComplete) {
    return (
      <FastTrackOnboarding
        onComplete={handleOnboardingComplete}
        onSkip={handleOnboardingSkip}
      />
    );
  }

  return <ChatScreen />;
}

Progressive Data Collection During Chat

How It Works

During conversation with AI:

Plain Text


User: "I love watching football"
AI: "That's great! What's your favorite team?"
User: "The Seahawks"
AI: [Records this to progressive data]



In your chat component:

TSX


// When user mentions something new
const recordProgressiveData = async (userId, data) => {
  await fetch(`/api/onboarding/progressive-data/${userId}`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(data)
  });
};

// Example: User mentions favorite team
recordProgressiveData(userId, {
  favoriteTeam: "Seattle Seahawks"
});
